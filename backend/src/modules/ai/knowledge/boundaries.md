# TMS — Features That Do NOT Exist

## Invoices
- The Invoices page is a PLACEHOLDER. It shows an empty page.
- There is NO "New Invoice" button.
- There is NO invoice creation, editing, or deletion.
- There is NO way to generate invoices from orders.
- There is NO SmartBill integration active.
- Orders do NOT have an Invoices tab or any invoice-related functionality.
- IMPORTANT: The "Preview PDF" button on orders generates a **shipping order**, NOT an invoice. If a user asks about "invoices" or "the PDF", clarify that the PDF is a shipping order, not an invoice.
- If asked about invoices, say: "The Invoices module is not yet available. It is planned for a future update. The PDF you can preview or send from an order is a shipping order, not an invoice."

## GPS Tracking
- There is NO real-time GPS tracking of vehicles.
- There is NO route planning or map integration.
- There is NO way to track vehicle locations on a map.

## Mobile App
- There is NO mobile app for TMS.
- TMS is a web-only application accessed through a browser.

## Features That Do Not Exist
- There is NO multi-language UI toggle (the interface is in English only).
- There is NO customer portal (partners cannot log in).
- There is NO automated scheduling or dispatch optimization.
- There is NO inventory or warehouse management.
- There is NO fuel card integration.
- There is NO driver management module (driver name is just a text field on each order).
- There is NO reporting module beyond the Statistics page.
- There are NO notifications or alerts system.
- There is NO bulk import of orders, partners, or vehicles from Excel/CSV.

## Statistics Limitations
- Profit predictions are **AI estimates**, not guarantees of future performance.
- Predictions require historical order data — if the database has no orders, no predictions are generated.
- There is NO export or download of statistics data (no CSV, no PDF from charts).
- There are NO custom date ranges — only preset periods (Day, Week, Month, Year, All Time).
- There is NO period comparison feature ("this year vs last year" is not available).
- Statistics data refreshes automatically when orders are created, updated, or deleted.

## What the AI Assistant Cannot Do
- I cannot create, edit, or delete orders, partners, or vehicles.
- I cannot access or modify any data in the system.
- I cannot perform any actions in TMS — I can only answer questions.
- I can only answer questions about how to use the TMS application.
- I cannot help with topics outside of TMS (politics, weather, general knowledge, etc.).
